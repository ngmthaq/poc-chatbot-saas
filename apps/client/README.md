# client

React 18 + Vite 7 + TypeScript frontend for the Call Center Agent platform.

## Stack

- **React 18** + **TypeScript** (strict)
- **Vite 7** — build tool and dev server
- **TanStack Router** — file-based routing
- **TanStack Query** — server state
- **MUI v6** — component library (light mode)
- **Axios** — HTTP client
- **Jotai** — global state

## Setup

```sh
# from workspace root
cp apps/client/.env.example apps/client/.env.local
# fill in VITE_API_BASE_URL
pnpm install
```

## Development

```sh
pnpm --filter client dev
```

Dev server runs on http://localhost:5173.

On first run, Vite generates `src/routeTree.gen.ts` automatically. Do not edit this file manually.

## Build

```sh
pnpm --filter client build
```

## Atomic Design

Components live in `src/components/` with strict import rules:

| Layer    | Folder       | May import from                 |
| -------- | ------------ | ------------------------------- |
| Atom     | `atoms/`     | external libs only              |
| Molecule | `molecules/` | atoms, external libs            |
| Organism | `organisms/` | molecules, atoms, external libs |
| Template | `templates/` | organisms, molecules, atoms     |
| Page     | `pages/`     | templates (+ stores, hooks)     |

## Environment Variables

| Key                 | Description                  |
| ------------------- | ---------------------------- |
| `VITE_API_BASE_URL` | Base URL for the server API  |
| `VITE_LIVEKIT_URL`  | LiveKit server WebSocket URL |
